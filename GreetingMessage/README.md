# 인사 메시지

같은 이름의 `Message` 클래스가 서로 다른 네임스페이스에 존재할 때, 타입 별칭을 사용하여 구분하는 프로그램을 작성하세요.

**요구사항**

1. `Formal` 네임스페이스에 `Message` 클래스 정의
   - `SayHello(string name)` 메서드: "[name]님, 안녕하십니까!" 출력
   - `SayBye(string name)` 메서드: "[name]님, 안녕히 가십시오!" 출력

2. `Casual` 네임스페이스에 `Message` 클래스 정의
   - `SayHello(string name)` 메서드: "안녕, [name]!" 출력
   - `SayBye(string name)` 메서드: "잘 가, [name]!" 출력

3. 메인 프로그램에서:
   - 타입 별칭을 사용하여 각 `Message` 클래스를 구분
   - 각 메시지 객체를 생성하고 메서드 호출

## 예상 실행 결과

```
=== 인사 테스트 ===
[격식체]
홍길동님, 안녕하십니까!
홍길동님, 안녕히 가십시오!

[비격식체]
안녕, 철수!
잘 가, 철수!
```
